CKEditor Accessibility Auditor
==============================

Provides a button in CKEditor which, when clicked, will run the
HTML_CodeSniffer Accessibility Auditor on the source code of the
content currently in the editor.

By default, "//squizlabs.github.io/HTML_CodeSniffer/build/" is used.
This base URL can be overridden in the CKEditor plugin settings.

For more information about HTML_CodeSniffer, see:
http://squizlabs.github.io/HTML_CodeSniffer/
