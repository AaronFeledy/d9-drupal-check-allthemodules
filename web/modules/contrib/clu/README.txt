INTRODUCTION
------------

This Module provides a feature to the Admin, So that admin can forcefully
logout to the currently logged user in the site.
for more info visit https://www.drupal.org/project/clu

REQUIREMENTS
------------

This module does not have any dependency.

INSTALLATION
------------

* Install as you would normally install a contributed Drupal module. See:
  https://www.drupal.org/documentation/install/modules-themes/modules-8
  for further information.

CONFIGURATION
-------------

* Copy the "clu" directory under "modules/";
* Go to "admin/modules" and enable "clu" module;
* Go to "admin/people/clu" list out currently logged user to the site with
  additional user info, with an option to forcefully logout the user.
