INTRODUCTION
------------

The QR Code Field Formatter will encode a specified field as a QR Code

REQUIREMENTS
------------

This module requires the following library.

 * PHP QRCode Library (https://github.com/aferrandini/PHPQRCode)

Add the required module to your composer.json file:

```shell
 {
    "require": {
        ...
        "aferrandini/phpqrcode": "1.0.1"
        ...
    }
}
```
Then run the command `composer update`.

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. See:
   https://www.drupal.org/node/1897420 for further information.
