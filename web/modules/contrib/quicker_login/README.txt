Quicker Login Module


INTRODUCTION
============

The Quicker Login module is a developer module that once enabled allows you to view any page as any user or login as any user.


REQUIREMENTS
============

No special requirements.


INSTALLATION
============

Install as you would normally install a contributed Drupal module. See: https://www.drupal.org/documentation/install/modules-themes/modules-8


USAGE
============

To log in as any user, visit: `/user/ql/{user_name}`

To visit a page as any user e.g. the content listing, visit: `/admin/content?ql={user_name}`

Note: While the module is enabled a warning message will be displayed on the site.
