Textarea Limit Module


INTRODUCTION
=============

The Textarea Limit module adds in character limiting to selected textarea widgets. Widgets are selected on the respective form display edit form(s). A global character limit can be used, configurable via ad admin form or a local limit can be set.


REQUIREMENTS
============

This module requires the jquery.limit javascript library which can be found at:
https://code.google.com/archive/p/jquery-limit/downloads. At the time of commit, version 1.3 is required which can be found at http://jquery-limit.googlecode.com/files/jquery.limit-1.3.js


INSTALLATION
============

Install as you would normally install a contributed Drupal module. See: https://www.drupal.org/documentation/install/modules-themes/modules-8

The jquery.limit js library file must be stored at /libraries/limit/jquery.limit-1.3.js
