NEXT PREVIOUS POST VIEW module for Drupal
---------------------------

DESCRIPTION
-----------
  Offering NEXT PREVIOUS POST VIEW without going to the listing page.


INSTALLATION:
-------------
  1. Extract the tar.gz into your 'modules' or directory and copy to modules
     folder.
  2. Go to "Extend" after successfully login into admin.
  3. Enable the module at 'administer >> modules'.

REQUIREMENTS
------------
  The basic nextpre module has node dependencies, nothing special is required.


CONFIGURATION
-------------
  The configuration page is at admin/config/nextpre,
  where you can configure the nextpre module.

UNINSTALLATION
--------------
  1. Disable the module from 'administer >> modules'.
  2. Uninstall the module

INTRODUCTION
-------------------------------------------
1. Install module “Next and previous link”.
2. Go to the “Block Layout”. Eg:- Admin Menu >> structure >> block layout
3. Go to the your block region and click on “Place order” and enable.
4. Go the “admin/config/nextpre” and set your content type name.
