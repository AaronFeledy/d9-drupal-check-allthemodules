Tacjs
================
Comply to the european cookie law is simple with *Tacjs*.

DEPENDENCIES
------------

* Tarteaucitron module.
  https://github.com/AmauriC/tarteaucitron.js/

CONFIGURATION
-------------

1. Download  Tarteaucitron  and rename directory to tarteaucitron and place it in the
   libraries folder (/libraries)')

1. Enable Tacjs in:
       admin/modules

2. You'll now find a Tacjs tab in
   administration page available at:
       /admin/config/tacjs/adminsettings
