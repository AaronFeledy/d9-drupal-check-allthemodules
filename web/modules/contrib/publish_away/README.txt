INSTALLATION
==============================================================================

1. Add these libraries packages to your composer.json file :

	"facebook/php-sdk-v4" : "4.0.*",
	"guzzle/plugin": "3.9.2"
    ~$ composer require "facebook/php-sdk-v4 4.0.*" "guzzle/plugin 3.9.2"

2. Download and install the Publish Away module and enable it.

3. Visit "Administration > Configuration > Web services > Publish away your contents" 
   and and complete default information about your accounts.
    
4. Create a new publish away field.
