INTRODUCTION
------------

This module has limited user registrations.for more info please ]
visit https://www.drupal.org/project/user_registration_limit

REQUIREMENTS
------------

This module does not have any dependency.

INSTALLATION
------------

* Install as you would normally install a contributed Drupal module. See:
  https://www.drupal.org/documentation/install/modules-themes/modules-8
  for further information.

CONFIGURATION
-------------

* Copy the "User Registration Limit" directory under "modules/" or "sites/all/modules";
* Go to "admin/modules" and enable "User Registration Limit" module;
* Go to "/admin/config/people/accounts" and configure the settings.
