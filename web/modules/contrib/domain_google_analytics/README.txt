
Module: Multidomain Google Analytics
Author: Sushil Pal <https://www.drupal.org/u/sushilpal>


Description
===========
Adds the Google Analytics tracking system to your multidomain website.

Requirements
============

* Google Analytics user account


Installation
============
Download and copy the module in to module directory.


Configuration / Usage
=====
All configuration is handled via the main settings page:
      admin/config/system/multidomain-google-analytics

All pages will now have the required JavaScript added to the
HTML footer can confirm this by viewing the page source from
your browser.
