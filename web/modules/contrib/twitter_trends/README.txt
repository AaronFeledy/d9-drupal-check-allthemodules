SUMMARY - Twitter Trends
========================
Twitter Trends Module provide the simple way to display the current trending
topics based on user's location into a block.


REQUIREMENTS
-------------
All dependencies of this module are enabled by default in Drupal 8.x.


INSTALLATION
-------------
Install this module as usual. Please see
http://drupal.org/documentation/install/modules-themes/modules-8

Module requires the Twitter Auth library if you downloaded from Drupal.org 
composer require abraham/twitteroauth

You can also download the module & its dependency by using composer
composer require abraham/twitter_trends


USAGE
-------
This module create 'Twitter Trends' block:

* Filter the number of trends topic from block configuration page.


CONFIGURATION
--------------
Global module settings can be found at /admin/config/twitter_trends/settings.

 * Register Twitter Application at https://apps.twitter.com.
 * Add the Key/Secret/Token with the one which has auto generated by twitter
  application page.
