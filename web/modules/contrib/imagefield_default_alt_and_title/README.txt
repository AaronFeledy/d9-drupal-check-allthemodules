INTRODUCTION
------------

The module "Imagefield default alt & title" adds alts and titles for images
to where they are not, taking them out of the title material.


INSTALLATION AND CONFIGURATION
------------------------------

0 - Prerequisites:
Requires the entity module to be enabled.

1 - Download the module and copy it into your contributed modules folder:
[for example, your_drupal_path/sites/all/modules] and enable it
from the modules administration/management page (requires the entity module to be
enabled).
More information at: Installing contributed modules (Drupal 7) [18].
