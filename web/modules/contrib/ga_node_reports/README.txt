INTRODUCTION
------------

Google Analytics Node Reports is an add-on module. That provides site
administrators and programmers to graphically represents the reports
based on Google Analytics data. This graphical representation of reports
is based on each node in Drupal system.

 * For a full description of the module, visit the project page:
   https://drupal.org/project/ga_node_reports

 * To submit bug reports and feature suggestions, or to track changes:
   https://drupal.org/project/issues/ga_node_reports

REQUIREMENTS
------------

 * Charts:
 https://www.drupal.org/project/charts
 * Google Analytics:
 https://www.drupal.org/project/google_analytics
 * Google Analytics Reports:
 https://www.drupal.org/project/google_analytics_reports

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit:
   https://drupal.org/documentation/install/modules-themes/modules-8
   for further information.

CONFIGURATION
-------------

  1. Go to config page (admin/config/services/google-analytics-reports-api)
     and Complete the fields that are displayed with the Google Analytics API.

  2. Go to config page (admin/config/system/google-analytics)
     and complete the fields that are displayed with the Google Analytics ID.

  3. You can see the analytics of a node, from the tab of the node or through
     the following URL (node/{nid}/analytics).
