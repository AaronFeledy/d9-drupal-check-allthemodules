Instructions for browser-refresh

  1. Installation

    You need nodeJS to be installed and version 4.2 or higher is required. Please refer to the nodeJS documentation for your operating system.

    The dependencies get installed and updated automatically when you start browser-refresh, see below.

  2. Start browser-refresh

    drush browser-refresh-start

  3. Stop browser-refresh

    drush browser-refresh-stop

  4. Get browser-refresh status

    drush browser-refresh-status

  5. Restart browser-refresh

    drush browser-refresh-restart
