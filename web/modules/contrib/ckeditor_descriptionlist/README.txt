CKEditor Description list
==============================

Description
===========
The Description list plugin provides the ability to create a definition list.

Usage
=====
Go to the Text formats and editors settings (/admin/config/content/formats) and
add the buttons : Tag dl, Tag dd, Tag dt to any CKEditor-enabled text format you want.

Installation
============

This module requires the core CKEditor module.

1. Download the plugin from https://github.com/Reinmar/ckeditor-plugin-descriptionlist
2. Place the plugin in the root libraries folder (/libraries).
3. Enable CKEditor Description list module in the Drupal admin.
4. Configure your WYSIWYG toolbar to include description list buttons.

Uninstallation
==============
1. Uninstall the module from 'Administer >> Modules'.
