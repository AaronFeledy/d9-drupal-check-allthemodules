---SUMMARY---

This module allows you to send warnings and more serious errors in the error
log via email to a specified address. Optionally, every PHP entry can be sent,
since this is definitely a problem on the site.


---INTRODUCTION---


Go to `/admin/config/development/logging` and enter the e-mail address that 
should receive copies of errors.


---REQUIREMENTS---


None.


---INSTALLATION---


Install as usual. Place the entirety of this directory in the /modules 
folder of your Drupal installation. 


---CONFIGURATION---


Set up the right behavior for you on admin/config/development/logging.
