CKEditor Equation Editor -  is an extension to the Drupal 8 CKEditormodule.

This module integrates the Equation Editor library into your Drupal site.
Equation Editor is the modern JavaScript based LaTex rendering solution for the
internet.


REQUIREMENTS
============
- ckeditor


FEATURES
========

This Equation Editor provides a toolbar with common mathematical operators and
symbols, to help write stunning equations using the LaTeX markup. A high
resolution preview of your equation is shown within the editor, which is
inserted into your document when you click 'Ok'.

This editor is power by the CodeCogs Equation Engine, the world's leading
equation rendering web-service.


INSTALLATION
============
1- Copy the ckeditor_eqneditor folder to your modules directory.
2- Download eqneditor library from http://ckeditor.com/addon/eqneditor.
3- Copy the downloaded content that is eqneditr folder in the root libraries
folder (/libraries).
4- Go to admin/modules and enable the module.
5- Go to admin/config/content/formats and configure the desired profile.
6- Move a button into the Active toolbar.
7- Clear your browser's cache, and a new button( fx ) will appear in your
toolbar.
