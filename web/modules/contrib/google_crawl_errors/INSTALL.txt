1. Download Google API library from https://github.com/google/google-api-php-client or use composer.
2. Place the google_crawl_errors module in your Drupal installation's modules folder.
3. Enable the module.
4. Go to /admin/config/services/google-crawl-errors
5. Set the "Site ID" field. You can name this to what ever you want. Ideally it should be just lower case letters and numbers. No spaces.
6. Set the "Site Url" field. (Including protocol and trailing slash. Example: https://www.example.com/ 
7. If you already have an OAuth 2.0 client credential you can skip to step 18. Otherwise, go to https://console.developers.google.com and make sure you are sign in.
8. Search for "Google Search Console API", click "Enable".
9. Click "Create Credentials"
10. In "Where will you be calling the API from?" field, select "Web browser (Javascript)".
11. In "What data will you be accessing?" field, select "User data".
12. In "Name" field, specify an appropriate name for your credential.
13. Enter your site's URI in the "Authorised JavaScript origins" field.
14. In the "Authorised redirect URIs" field, enter "https://YOURDOMAIN/google-crawl-errors/oauth".
15. Click "Create OAuth client ID".
16. In the "Set up the OAuth 2.0 consent screen" "Product name shown to users" field, give it an appropriate name. For example, "Search Console API".
17. Click "Continue".
18. Click the "Download JSON" icon, copy the file content to the "Google OAuth Secret JSON" field.
19. Under the "Google OAuth Token JSON" field, click the "OAuth redirect" URL link. Follow Google's on screen instruction. Copy the result JSON to the field and click save.
