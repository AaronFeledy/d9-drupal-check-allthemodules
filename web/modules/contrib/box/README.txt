INSTALLATION
------------

Installation and usage:
1. Install Box module
2. Enable Insert boxes text filter for desired text formats
3. (optional) Create new box bundles at Administration -> Structure -> Box types
4. Create box entities at Administration -> Content -> Boxes
5. (optional) Override box template to your liking
6. Embed boxes using text filter using:
   - entity ID: [box:entity_id] or [box:entity_id:view_mode]
   - entity machine name: [box-name:entity_machine_name] or [box-name:entity_machine_name:view_mode]
