MESSAGE TIME
----------


INTRODUCTION
------------
	This module message time delay.


INSTALLATION
------------


CONFIGURATION
-------------
	Goto: /admin/config/user-interface/message-time
	- Message Time
		Now the message box will disappear after 10 seconds, as we hardcoded the duration (10000ms = 10 sec)
