Search API Hashtag
------------------

Search API hashtag module provides an index field to store hashtag that appears
in one or more indexed fields. This module is useful to create a facet field to
filter indexed contents by their hashtags.

INSTALLATION
------------

* Install as usual, see http://drupal.org/node/70151 for further information.

USAGE
------------

* Create a Search API index in admin/config/search/search-api/add-index
* Go to Index Fields configuration tab and add a new field
* In General fields group the field Hashtag will be displayed, add it
* Configure the Hashtag field to set which content fields will have hashtags
* New Hashtag field will be available in the Index field list
