---SUMMARY---

Notify users if a new revision is saved when editing a moderated node.

For a full description visit project page:
https://www.drupal.org/sandbox/mogtofu33/2861988

Bug reports, feature suggestions and latest developments:
https://www.drupal.org/project/issues/2861988


---REQUIREMENTS---


Drupal core module Content moderation.


---INSTALLATION---


Install as usual.

Configure from /admin/config/system/content_moderation_edit_notify
