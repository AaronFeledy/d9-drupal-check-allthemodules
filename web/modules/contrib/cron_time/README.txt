INTRODUCTION
------------

This simple Drupal 8 module using for custom Cron Time set.

REQUIREMENTS
------------

 -No any special requirements.

INSTALLATION
------------

Install via /admin/modules
drush en cron_time -y
composer require drupal/cron_time

CONFIGURATION
-------------

Goto: /admin/config/system/cron
