INTRODUCTION
------------

Modal Blocks module displays a blocks in as popup in a page.

INSTALLATION
------------

1.Download Modal Blocks module from https://www.drupal.org/project/modal_blocks
  and enable the module.
2.Goto Structrue > Block layout > place Modal block in any region.

CONFIGURATION
-------------

The module has no menu or modifiable settings. There is no configuration.
