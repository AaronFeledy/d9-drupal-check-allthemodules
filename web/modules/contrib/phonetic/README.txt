SUMMARY
-------

This module provides a input filter to mask profanity in user input, it filters
out words based on their phonetic key.

INSTALLATION
------------

Normal Drupal module installation, for more information
see http://drupal.org/documentation/install/modules-themes/modules-7

CONFIGURATION
-------------

Enable this input filter for any desired input format (see Configuration >
Content authoring > Text formats).
Use the admin configuration page located at admin/settings/content/phonetic
to configure blacklist, whitelist, and other general settings.
See the included WORDS.txt file for an example set of words use as the
blacklist.
