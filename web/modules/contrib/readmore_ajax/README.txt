INTRODUCTION
------------

The Read more ajax module is a simple module which adds a Read more ajax link to
teaser. This will in effect create a blog like on facebook.

 * For a full description of the module visit:
   https://www.drupal.org/project/readmore_ajax

 * To submit bug reports and feature suggestions, or to track changes visit:
   https://www.drupal.org/project/issues/readmore_ajax


REQUIREMENTS
------------

This module requires no modules outside of Drupal core.


INSTALLATION
------------

Install the Read more ajax module as you would normally install a contributed
Drupal module. Visit https://www.drupal.org/node/1897420 for further
information.


CONFIGURATION
-------------

    1. Navigate to Administration > Extend and enable the Read more ajax module.
    2. Navigate to Administration > Configuration > System > Read more ajax and
       select node types you want to activate ajax readmore on. If nothing is
       selected, AJAX will be enabled everywhere.
