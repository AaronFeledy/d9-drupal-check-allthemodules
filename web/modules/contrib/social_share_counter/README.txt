INTRODUCTION
------------

The Social Share Counter module enables a extensible free Share Button ecosystem
for the best possible social sharing optimization of your website.

 * For a full description of the module visit:
   https://www.drupal.org/project/social_share_counter

 * To submit bug reports and feature suggestions, or to track changes visit:
   https://www.drupal.org/project/issues/social_share_counter


REQUIREMENTS
------------

This module has no dependency on any other module.


INSTALLATION
------------

Install the Social Share Counter module as you would normally install a
contributed Drupal module. Visit https://www.drupal.org/node/1897420 for further
information.


CONFIGURATION
-------------

    1. Navigate to Administration > Extend and enable the module and its
       dependencies.
    2. Set the access Social Share Counter permissions for authenticated users
       at Administration > User > Permissions.
    3. Navigate to Administration > Configuration > System > Social Share
       Counter for basic configurations.
    4. The General Setting field group allows the user the enable/disable
       display buttons, edit the text below the count, and set a minimum count
       to show.
    5. The Button Options field group allows the user to change the text on the
       buttons for Facebook and Twitter.
    6. The Placement field group allows the user to choose which Node Views
       modes to which display the counter. There is also and optional weight
       value which determines the location on the page where it will appeared in
       the content section.
    7. Save Configurations.

Features:
 * No images used- All social media icons are fontawesome fonts.
 * Social Share Counter.
 * Responsive design.
 * Social Share Sticky Top Bar.
 * Options to enable/disable with node types with different view modes the user
   wishes to display.
 * Available as a block.
