INTRODUCTION
------------

Provides a Weather Forecasts Block.

 * For a full description of the module, visit the project page:
   https://drupal.org/project/weather_city

 * To submit bug reports and feature suggestions, or to track changes:
   https://drupal.org/project/issues/weather_city

REQUIREMENTS
------------

This module no require additional modules.

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit:
   https://drupal.org/documentation/install/modules-themes/modules-8
   for further information.

CONFIGURATION
-------------

Create a widget in https://www.tutiempo.net/gratis.htm.

Example code generated by the web tutiempo.net:

<!-- www.tutiempo.net - Ancho:477px - Alto:91px -->
<div id="H3LL0FRI3ND">
El tiempo - Tutiempo.net
</div>
<script type="text/javascript" 
src="https://www.tutiempo.net/s-widget/H3LL0FRI3ND"></script>

Set the block with the following parameters:

 * Weather ID: H3LL0FRI3ND.
 * Weather Link: https://www.tutiempo.net/s-widget/H3LL0FRI3ND.
