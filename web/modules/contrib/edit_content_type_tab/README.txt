Drupal Edit Content Type Tab Module module:
------------------------------------------
Maintainers:
  Ian Flory (http://drupal.org/user/1224098)
Requires - Drupal 8
License - GPL (see LICENSE)

Overview:
--------
Presents users with a tab to edit the content type of the current node.

Installation:
------------
1. Place the Edit Content Type Tab Module directory in your
   modules folder (this will usually be "modules/").
2. Go to "Extend" and enable the module.
